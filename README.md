# AI Code Reviewer

An AI-powered web application that analyzes source code and provides automated code reviews.

## Features
- Paste code and receive instant AI feedback
- Detect bugs and bad coding practices
- Suggest improvements and optimizations
- Simple and lightweight interface

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express  
AI: OpenAI API  
Deployment: Vercel

## Live Demo
https://ai-code-review-39a1.vercel.app/

## How It Works
1. User pastes code into the input box
2. Code is sent to backend API
3. Backend sends prompt to LLM
4. AI analyzes the code
5. Suggestions are returned and displayed
